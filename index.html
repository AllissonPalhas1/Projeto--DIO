História da WEB

A 1ª geração de computadores surgiu no ano de 1944, chamado de Harvad Mark 1 , era enorme e ocupava um andar inteiro de um prédio. Logo depois surgiu o Colossus em 1946, ele era bem menor que Mark 1. Colossus foi criado pela Alemanha na segunda guerra mundial, onde ele era usado para fazer cálculos relacionados as táticas de guerras. No mesmo ano surgiu o ENIAC, criado pelos Estados Unidos. Diferente do Mark 1 que era eletromecânico, o ENIAC ja era mais automático. Uma curiosidade é que na mesma epóca (1946), surgiu o termo "Bug", pois como os computadores esquentavam tanto, que atraiam insetos, causando falhas. A 2ª geração de computadores surgiram entre os anos (1959-1965), com o surgimento dos transistores, onde o uso de válvulas deixou de ser usados e substituídos pelos transistores, diminuindo imensamente o tamanho dos computadores. A 3ª geração de computadores surgiram entre (1965-1970), com os circuitos integrados: transistor, capacitores e resistores, diminuindo ainda mais de tamanho os computadores. A 4ª geração de computadores surgiram a partir de (1971), onde são conhecidos atualmente os circuitos intergrados passaram para microprocessadores, chips atualizados, etc. Uma curiosidade dessa geração, foi a criação do primeiro mouse, pela Apple, uma grande inovação da época. A 5ª geração, é o que está por vir ...

Client x Server

Client seria a forma que os usuários utilizam diferentes tipos de navegadores para encontrar conteúdos que os interessam, conectados a internet. Já os Servers, são lugares onde são armazenados dados, tanto locais (server da própria empresa), quantos nos servers que são definidos para a utlização de qualquer usuário. Servers locais, podem ser acessados off-line.

Linguagens de Programação

Definição:

São conjuntos de dados lógicos que especificam uma determinada idéia/ação para criação de Softwares.

Server-Side:

Agumas linguagens que são usadas -> php, Ruby, C#, JS e Java.

Client-Side:

A principal linguagem para quem ta começando na área da Programação -> JS. Uma curiosidade a respeito dessa linguagem, é que ela roda tanto no Client-Side, quanto no Server-Side, pois nos servers existem um módulo chamado V8 que serve para interpretar o JS, entre o Client e server, mais para o V8 funcionar no caso dos Server-Side é preciso a instalação de um programa chamado Node js.

HTML

Definição:

HTML: é uma linguagem de marcação de Hiper Texto, (HyperText Markup Language). É uma linguagem utilizada para a criação de pãginas Web. Navegadores as interpretam e explicitam o conteúdo da página ao client.

Estrutura Básica:

Básicamente é formado pelo Tipo de Doc, que seria = HTML; Pelo Head que seria o cabeçalho, por exemplo o tìtulo da página; e pelo Body que seria toda a informação da página exposta/visível ao usuário.

Tags/Atributos das tags:

Básicamente as tags são comandos que realizam personalizações no conteúdo indicado pelo programador. Alguns exemplos de tags são: hr > serve para criação de uma linha horizontal logo abaixo do texto; i > serve para deixar o text em itálico; Mark > serve para marca o texto; U > serve para sublinhar o text; Strong > serve para deixar o text em negrito, imput > cria um campo na página para digitação do usuário; style > edita a cor do text; img > anexar uma imagem na página web através de um endereço (link); id > seria uma tag definidora de uma variável, que se for editada, consequentemente todas as outras também seram alteradas; Blockquote > serve para citações de texto; sub > é definida para o texto subscrito; sup > serve para elevar o texto acima do texto; entre outras tags.. Os atributos das tagas são o valor definido pela tag, por exemplo a tag style onde o atributo dela seria a cor da váriavel, outro exemplo, seria o width > defini a largura da imagem (img). Lembrando que todas as tags ao ser aberta é necessário seu fechamento, porém algumas ao ser aberta ja é fechada automáticamente pelo próprio sistema, caso do imput.

Textos:

Os texto no HTML são toda forma escrita visível ao usuário da página. São usados as funções (h1 até h6), definidas como titulo, subtitulo, e assim em adiante até o h6; onde esses titulos dependeria do h > (h1, h2, h3, h4, h5, h6) que o criador da página optar. Lembrando que cada tipo tem definida uma fonte da maior á menor.

Listas Ordenadas e Listas Não Ordenadas

As listas ordenadas são aquelas organizadas numéricamente, ou seja, listadas por números. Já as listas não ordenadas são aquelas organizadas por símbolos, ou seja, listadas por pequenos círculos pretos ao lado do texto.

Tag Link

A tag link é definida pelo local/texto que, quando o usuário clicar, será direcionado ao quer o programador definir, por exemplo a outra página do site da empresa, a outro site de fora, a outro tópico da página, etc.
